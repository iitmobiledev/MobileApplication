chart
directive in module myApp

Description

Директива добавляет на страницу приложения график

Usage

as element:
<chart
       chart-data="{String}">
</chart>

Parameters

ParamTypeDetails
chart-dataString

имя параметра со списком данных для отображения графика: массив из списков, где 1 элемент-объект {Data}, 2 элемент-величина в зависимости от типа графика

Methods

  • drawChart()

    Метод, рисующий график

orient
directive in module myApp

Description

Директива задает ориентацию экрана устройства

Usage

as element:
<orient
       orientation="{String}">
</orient>

Parameters

ParamTypeDetails
orientationString

принимает следующие значения: 'landscape' - ландшафтная ориентация экрана, 'portrait' - портретная ориентация экрана, 'any' - автоматическое определения ориентации экрана устройством

DateHelper
service in module myApp

Description

Сервис для работы с датами. Позволяет получить предыдущую дату с помощью метода getPrev, получить период методом getPeriod, получить возможные шаги полем steps.

Methods

  • getMonthTitle(monthNumber)

    Метод для получения названия месяца по его номеру

    Parameters
    ParamTypeDetails
    monthNumberNumber

    номер месяца, начиная с 0

    Returns
    String

    название месяца

  • getPeriod(date, step)

    Метод предназначен для получения периода, т.е. начальной даты и конечной даты.

    Parameters
    ParamTypeDetails
    dateDate

    дата, по которой будет определяться период.

    stepString

    шаг, показывающий какой период необходимо вернуть, должен быть определен в DateHelper.steps.

    Returns
    Period

    объект с полями {Date} begin и {Date} end, обозначающими начальную и конечную даты периода.

  • getPrev(date, step)

    Метод предназначен для получения того же дня на прошлой неделе или прошлой недели, или прошлого месяца. Необходимое указывается параметром step.

    Parameters
    ParamTypeDetails
    dateDate

    дата, для которой будет вычислена предыдущая дата.

    stepString

    шаг, показывающий за какой период необходимо вычислить предыдущую дату.

    Returns
    Date

    предыдущая дата.

headerContent

Description

Директива для хедера

Usage

as element:
<header-content
       title="{String}"
       show-back-button="{Boolean}">
</header-content>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
titleString

Заголовок окна, отображаемый в хедере

show-back-buttonBoolean

Указание необходимости отображения кнопки "Назад"

footerContent
directive in module myApp

Description

Директива добавляет на страницу приложения навигационную панель (футер), содержащую кнопки переходов на страницы отчетности, визитов и настроек

Usage

as element:
<footer-content
       show="{Boolean}">
</footer-content>

Parameters

ParamTypeDetails
showBoolean

если true - показывает навигационную панель(футер), иначе - скрывает

SettingsController
controller in module myApp

Description

Контроллер, отвечающий за загрузку данных о текущем пользователе.

$scope содержит следующие поля:

  • user - текущий пользователь,
  • exit - функция для выхода пользователя из системы и перехода на страницу авторизации.

Dependencies

  • UserLoader

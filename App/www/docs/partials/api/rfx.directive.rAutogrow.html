rAutogrow
directive in module rfx

Description

Директива добавляет на страницу приложения хедер, содержащий заголовок, и если указано, кнопку перехода на предыдущую страницу

Usage

as element:
<r-autogrow
       title="{String}"
       show-back-button="{Boolean}">
</r-autogrow>

Parameters

ParamTypeDetails
titleString

Заголовок окна, отображаемый в хедере

show-back-buttonBoolean

Указание необходимости отображения кнопки "Назад"

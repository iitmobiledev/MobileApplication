headerContent
directive in module myApp

Description

Директива добавляет на страницу приложения хедер, содержащий заголовок, и если указано, кнопку перехода на предыдущую страницу

Usage

as element:
<header-content
       title="{String}"
       show-back-button="{Boolean}">
</header-content>

Parameters

ParamTypeDetails
titleString

Заголовок окна, отображаемый в хедере

show-back-buttonBoolean

Указание необходимости отображения кнопки "Назад"

Methods

  • show()

    Отображает хедер на странице. Пытается выполнится до тех пор, пока не будет подгружена библиотека intel.xdk.

  • show()

    Отображает кнопку назад на хедере, при условии, что атрибут showBackButton равен true.

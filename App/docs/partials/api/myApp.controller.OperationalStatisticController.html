OperationalStatisticController
controller in module myApp

Description

Контроллер, отвечающий за загрузку статистических данных.

$scope содержит следующие поля:

  • date - текущая дата,
  • step - текущий шаг периода, должен быть определен в DateHelper.steps,
  • hasPrevData - показывает, есть ли данные за предыдущий период,
  • hasFutureData - показывает, есть ли за следующий период.

Dependencies

  • OperationalStatisticLoader
  • DateHelper

Methods

  • hasFutureData()

    Метод для проверки наличия данных за будущий период.

    Returns
    Boleean

    Возвращает true, если есть данные за будущее.

  • hasPrevData()

    Метод для проверки наличия данных за прошлый период.

    Returns
    Boleean

    Возвращает true, если есть данные за прошлое.

  • toChart()

    Метод для перехода на страницу графика.

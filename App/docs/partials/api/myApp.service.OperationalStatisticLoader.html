OperationalStatisticLoader
service in module myApp

Description

Сервис для загрузки статистики за период.

Dependencies

  • DateHelper
  • OperatonalStatisticsDataSumming

Usage

OperationalStatisticLoader(date, step);

Parameters

ParamTypeDetails
dateDate

дата, за которую будут подгружаться данные.

stepString

шаг периода (этап, например, 'day'), за который будут получены данные, должен быть прописан в DateHelper.steps

Returns

OperationalStatistics

объект статистики

GraphicController
controller in module myApp

Description

Контроллер, получающий данные для отрисовки графика

Usage

GraphicController(type, period, data);

Parameters

ParamTypeDetails
typeString

тип графика(что отображает график)

periodNumber

период в месяцах, за который должен быть отрисован график

dataArray

данные для отображения графика:массив из списков, где 1 элемент-объект {Data}, 2 элемент-величина в зависимости от типа графика

Methods

  • changePeriod(p)

    Метод изменяющий период для отображения

    Parameters
    ParamTypeDetails
    pNumber

    Период, на который нужно изменить в месяцах

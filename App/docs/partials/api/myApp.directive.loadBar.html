loadBar
directive in module myApp

Description

Директива добавляет на страницу приложения анимацию загрузки(loaderBar)

Usage

as element:
<load-bar
       param-name="{String}">
</load-bar>

Parameters

ParamTypeDetails
paramNameString

имя параметра за которым нужно следить чтобы понять нужно ли отображать лоадбар.
